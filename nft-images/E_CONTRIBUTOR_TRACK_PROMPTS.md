# NFT Image Prompts — Category E: Contributor NFT Tracks (25 Items)

## Style Guide (Apply to ALL prompts below)

**Base style prefix** (prepend to every prompt):
```
Dark fantasy RPG achievement badge, centered on pure black background (#000000), no text, no watermark, dramatic lighting, highly detailed digital painting, ornate heraldic seal style with magical energy, progression from simple to divine
```

**Contributor NFTs are soulbound progression badges. Each track has 5 tiers (Common → Legendary). The visual progression should be obvious — same core motif escalating in complexity and power.**

**Resolution**: Generate at 512×512. Downscale to 64×64 for icons.
